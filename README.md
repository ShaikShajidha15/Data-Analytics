# Data-Analytics-Project
End-to-end data analytics projects using Excel, SQL, Python, and Power BI. Its covering data cleaning, exploratory analysis, visualization, and business insights.

---

**Customer Shopping Behavior Analysis**

**Project Overview**
This repository contains an end-to-end **Data Analytics project** focused on analyzing **customer shopping behavior** using **Python, SQL, and Power BI**.  
The project aims to extract meaningful insights from customer purchase data, identify patterns and trends, and present them through interactive visualizations to support data-driven decision-making.

---

**Dataset**
- File : customer_shopping_behavior
- Description : Contains customer demographic details, product history, purchase history, payment methods, product categories, and spending information.

---

**Tools & Technologies**
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI
- Jupyter Notebook
- CSV Dataset

---

**Repository Structure**
* customer_shopping_behavior.csv
* Customer_shopping_Behavior.Analysis.ipynb
* customer_behavior_sqlqueries.sql
* customer behaviour dashboard powerBI.pbix
* README.md

---

**Dataset Description**
The dataset contains customer shopping-related information such as:
- Customer demographics
- Purchase frequency
- Product categories
- Spending behavior
- Transaction details  

---

**Data Analysis (Python)**
The Jupyter Notebook includes:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Customer segmentation
- Purchase and spending pattern analysis
- Category-wise insights
- Visualization of trends and metrics

---

**SQL Analysis**
The SQL file contains queries for:
- Customer segmentation
- Revenue and sales analysis
- Category-wise performance
- Purchase frequency and behavioral insights

---

**Power BI Dashboard**
The Power BI dashboard provides:
- Interactive visualizations
- Key Performance Indicators (KPIs)
- Customer behavior trends
- Sales and category insights

Open the `.pbix` file using **Power BI Desktop** to explore the dashboard.

---

**How to Run the Project**
1. Create the repository
2. Open Customer_shopping_Behavior.Analysis.ipynb in Jupyter Notebook
3. Ensure the CSV file is in the same directory
4. Run the notebook cells step by step
5. Open the Power BI file using Power BI Desktop
